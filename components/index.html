Desktop Header (live component above)

The site header above this page is the live navigation component. It features: sticky positioning with scroll shadow, logo left-aligned, primary nav links centered with hover underline animations, and utility icons right-aligned (search, account, cart + subscribers CTA).

Dropdowns open on hover (desktop) with a smooth opacity + translateY transition. Mobile nav uses a slide-in drawer from the right with accordion-style section expansion. Nav collapses at ≤1024px viewport width.
